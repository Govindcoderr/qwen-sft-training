## Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd Qwen 2.5
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
